람다식(Lambda expression) --> 람다 표현식
함수형 프로그래밍과 람다식

- 자바: 객체지향 프로그래밍
- 기능을 수행하기 위해서 객체를 만들고 멤버변수, 메서드를 수행하게 코드를 설계한다.

@ 함수형프로그래밍방식
 - 함수의 구현과 호출만으로 프로그래밍방식을 지원하는 방식.(--> 람다표현식이라고 한다.)
 
@ 함수형 프로그래밍(FP, Functional Programming)
 - 함수형 프로그래밍은 순수함수(Pure Function)를 구현하고 호출함으로써
 외부 데이터에 부수적인 영향(Side-effect)을 주지 않도록 구현하는 방식.
 => 즉, 매개변수만을 사용하여 만드는 함수를 순수함수라고 함.??
--> 실제로 많이 사용하는 표현식이라서 배워둬야 한다..

@ 람다식 문법
- 익명함수 --> 람다식으로 발전
